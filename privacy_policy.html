General Information on Data Processing

I only collect and process personal data to the extent necessary and in accordance with the applicable data protection regulations, particularly the General Data Protection Regulation (GDPR). No cookies are used on my portfolio website.

Collection and Storage of Personal Data and Type and Purpose of Their Use

This website is hosted via GitHub Pages, a service provided by GitHub, Inc. (88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA). When visiting my website, GitHub automatically collects certain information, which is stored in so-called server log files.
This data includes:
  • IP address of the accessing device
  • Date and time of access
  • Browser type and version
  • Operating system used
  • Referrer URL (previously visited page)
    • This data is used solely to ensure the technical operation and security of the website. This data is not combined with other data sources. For more information on GitHub's data processing, please see GitHub's privacy statement: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

    Legal Basis for Processing

    The processing of server log files is based on Art. 6(1)(f) GDPR (legitimate interest). My legitimate interest lies in ensuring the secure and uninterrupted provision of my website.

    Storage Duration

    Server log files are stored by GitHub according to their own policies and are automatically deleted. I have no influence over this storage duration.
